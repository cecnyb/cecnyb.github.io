Bachelor's thesis

  • Accessible Board Game Playing: Report | Video
    The Bachelor's thesis aimed to make board games more accessible to people with visual impairment. The focus was on representing cards in a more accessible format through an application. The appliation uses a camera with image recognition to scan cards and match them with the corresponding card in the database. Then a digitial hand is stored in the application, which can be used to play the game. The application was tested with the board game Catan and the results showed that the application was able to recognize the cards with high accuracy.
